🔒 TLS Cert Monitor

Monitor Your TLS/SSL Certificates

A Go-based monitoring tool that scans directories for TLS/SSL certificates and exposes detailed Prometheus metrics for proactive security monitoring.

🚀 Get Started Learn More →

✨ Features

📁 Auto Certificate Discovery

Scans specified directories and automatically detects all TLS/SSL certificates.

📊 Prometheus Metrics

Exports expiration dates, key strengths, issuer details, and health checks.

⚡ High Performance

Optimized for large infrastructures with concurrency support for scanning.

⚙️ YAML Config

Configure scan intervals, workers, and monitored directories easily.

🔔 Expiration Alerts

Integrates with Prometheus Alertmanager to notify before certs expire.

🔐 Secure by Default

TLS/SSL enabled APIs with hardened defaults and minimal dependencies.

🚀 Installation

Go Install

go install github.com/your-org/tls-cert-monitor@latest

Docker

docker run -d \
  -v /etc/ssl:/certs \
  -p 8080:8080 \
  ghcr.io/your-org/tls-cert-monitor:latest

Kubernetes Deployment

kubectl apply -f https://raw.githubusercontent.com/your-org/tls-cert-monitor/main/deploy/k8s.yaml
